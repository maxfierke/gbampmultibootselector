GBAMP Multiboot Selector v2.0 by TeenDev

GBAMP Multiboot Selector requires 3 different loaders.

The first is 0xtob's
http://tobw.net/ds/devlauncher.zip (needs DLDI Patch for GBAMP).

DSOrganize
http://www.dragonminded.com/?loc=ndsdev/DSOrganize

Moonshell
http://mdxonline.dyndns.org/archives/nds/ (Rename 'moonshell.nds').

Key Assignments:

A: Oxtob's Loader
B: DSOrganize
X: MoonShell
Y: brings up a Boot menu to select which .nds file to boot.
Start: Boot into GBA Mode
